NASA 행성 위치 계산
✅ 내장 실시간 연산
실제로 하는 것:
NASA JPL이 공개한 DE430 케플러 궤도 방정식을 JavaScript로 앱에 내장 구현. 지금 이 순간의 율리우스 날짜를 기준으로 태양·달·수성·금성·화성·목성·토성 7행성의 황도 경도를 실시간 연산합니다. 앱을 열 때마다 현재 시각의 행성 위치가 자동 계산됩니다.

왜 "실시간"인가요?
외부 서버 호출 없이 앱 내부에서 현재 시각 기반으로 매번 새로 계산합니다. 구글 캘린더의 일출·일몰 계산도 동일한 방식(수식 내장)을 사용합니다. 일출/일몰이 정확한 것처럼, 행성 위치도 역학에 활용하기에 충분히 정확합니다.

정확도:
ELP2000(달) + VSOP87 간략판(행성) 공개 천문 계산식 기반. 오차 약 0.5° 이내 — 역학 활용(역행/순행 판별)에 충분한 정밀도.

솔직한 한계:
NASA JPL 서버에 직접 접속하는 실시간 API 호출 방식이 아닙니다. 극도로 정밀한 천문 관측(망원경 수준)과는 다릅니다. 그러나 역학 계산에는 0.5° 오차로도 완전히 충분합니다.
📡 VSOP87 + ELP2000 내장 · NOAA 태양풍(Kp지수)은 실시간 외부 API 병행

날씨·기상 데이터
🟢 실시간 API · 위치 자동 연동
실제로 하는 것:
Open-Meteo API로 현재 위치의 기온·습도·기압·풍속을 실시간 수신합니다.

위치 자동 연동 (4단계 우선순위):
1순위 — 정보 입력의 현재 위치 좌표 자동 사용
2순위 — 계산 후 저장된 현재 위치 좌표
3순위 — GPS 실시간 위치 (권한 허용 시)
4순위 — 서울 기본값 (아무것도 없을 때)

예: 부산 해운대 입력 → 부산 날씨 자동 수신 ✅
해외 여행 중 → 현지 날씨 자동 반영 ✅

역학 연동 방식:
기압 1013hPa 기준 편차 → 오행 에너지 보정.
고기압(맑음) → 양(陽) 에너지 강화, 저기압(흐림) → 음(陰) 에너지 강화.

한계:
위치 미입력 + GPS 미허용 시 서울 기본값 사용. 인터넷 미연결 시 기본값(기온 15°C·기압 1013hPa) 적용.
📡 open-meteo.com · 정보 입력 위치 자동 연동

실제로 하는 것:
스마트폰 자이로스코프·가속도계로 현재 방위각(0~360°)을 측정. DeviceOrientationEvent API 사용.

풍수 연동:
북(수行)·동(목行)·남(화行)·서(금行)·중(토行) 오행 방위 이론으로 현재 방향의 에너지 분석.

한계:
iOS 13+ 권한 허용 필요. 실내 자기장 간섭 시 오차 발생 가능.

📷 카메라 분석 (참고용)
보유 기술 스택:
스마트폰 전면 카메라로 얼굴 피부색의 미세 변화(혈류 파동)를 프레임별 분석. Butterworth 2차 밴드패스 필터(0.7~3.0Hz @20fps) 적용, RR 인터벌 피크 검출로 심박(BPM)·HRV 추출, FCI 안면혈색지수 실시간 계산.

과학적 근거:
원격 광용적맥파(rPPG) 기술은 MIT·스탠퍼드·서울대 등 수백 편의 국제 학술 논문으로 검증된 기술입니다. 조명이 밝고 움직임이 적을수록 정확도가 높아집니다.

현재 활용 방식:
CGO-FULI는 rPPG 기술을 역학 에너지 분석의 생체 보정값으로 활용합니다. 심박·HRV·혈색 데이터가 오행 에너지, NASA 행성 에너지와 융합되어 더 개인화된 ILI 점수를 만들어냅니다.

중요 안내:
의료기기가 아닙니다. 병원 진단·처방을 대체하지 않습니다. 건강 이상 의심 시 반드시 의료 전문가와 상담하세요. 본 측정값은 오행·역학 에너지 분석의 참고 데이터로만 사용됩니다.
